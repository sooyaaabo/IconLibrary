欢迎来到 **IconLibrary**！

本仓库内容为个人自用图标库，包括 **Emby**、**App** 和 **Flag** 图标等。

这些图标均来源于网络，并经过整理与分类，旨在便捷地提供各种图标供使用。

## 图标分类

1. **App** - 收录 App Store 等平台内的应用程序图标。

2. **Emby** - 专为 Emby 媒体服务器设计的图标集合。

3. **Flag** - 包含各国国旗及其他标志性旗帜图标。
